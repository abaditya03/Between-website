## 🔧 Customization Guide

### Easy Customizations (5-10 minutes)

**Change Colors:**
```javascript
// tailwind.config.js
colors: {
  'between': {
    'magenta': '#YOUR_COLOR',
    // ...
  }
}
```

**Update Testimonials:**
```typescript
// components/Testimonials.tsx
const testimonials = [
  { name: 'Your Name', quote: 'Your quote here' },
  // ...
]
```

**Modify Pricing:**
```typescript
// components/Pricing.tsx
```
