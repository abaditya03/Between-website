## 2024-04-05 - Redundant DOM Lookups in Interval Handlers
**Learning:** This codebase contained repeated `document.getElementById` lookups within a high-frequency `setInterval` loop for the SOS countdown UI logic. This architectural anti-pattern can significantly block the main thread by repeatedly traversing the DOM, reducing overall animation and execution smoothness.
**Action:** When working on JavaScript logic in static HTML templates within this architecture, always aggressively cache DOM elements in constant references at the start of the script block rather than retrieving them on the fly, particularly if they are accessed within timeouts or intervals. Ensure all event listeners are added via JS rather than inline attributes to keep HTML clean and avoid lookup repetition.
